# Steps:

1. Clone the rep
2. cd repo
3. `npm install`
4. `npm start`

open the link `http:localhost:3000`

# Layout

Currently app is divided in 3 parts: 

1. Product view/slider images
2. Select options/ Swatch elements
3. Grid views (to show views in Grid format)
